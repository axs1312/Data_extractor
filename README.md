# US Federal Agencies Data

A structured dataset of US federal agencies with contact and procurement information.

## Contents

`us_federal_agencies.csv` — 232 agencies with the following fields:

| Column | Description |
|---|---|
| Agency Name | Full official name |
| Acronym | Common abbreviation |
| Parent Agency | Parent department (if a sub-agency) |
| Official Website | Agency homepage URL |
| Address | Headquarters mailing address |
| Contact Phone | Main public phone number |
| Contact Email | General contact email |
| Headcount (Mar 2025 est.) | Estimated employee count as of March 2025 |
| Notes | Caveats on headcount or scope |
| Procurement Contact Name | OSDBU or procurement office contact |
| Procurement Contact Title | Title of procurement contact |
| Procurement Phone | Procurement office phone |
| Procurement Email | Procurement office email |
| OSDBU/Procurement URL | Link to small business / procurement page |

## Source

Data compiled from publicly available agency websites and the [SAM.gov](https://sam.gov) federal procurement database.
